== Description ==
Wettervorhersage widget is a free weather forecasting widget for your beautiful wordpress website.

* Supported in German and English
* The widget is responsive, perfect for mobile and desktop.
* Widget style like background and color can be customized
* No development skills are needed
* High performance and lightweight code
* SEO friendly

** For special customization requests please contact this mail -> wettervorhersage@wetter2.com

The Wettervorhersage plugin is relying on a 3rd party as a service, the widget is using weather services provided by wetter2.com.
* Service provider link - https://www.wetter2.com
* Service provider terms of use link - https://www.wetter2.com/terms
* Service privacy policy link - https://www.wetter2.com/privacy


== Installation ==

1. Upload `wettervorhersage.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the wettervorhersage widget to your page from the widgets tab
